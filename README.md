# A Higher or Lower Game made in React

Compare pokemon based on total stats.

[Live Demo](https://pokeguess.teraprath.dev)

### Install
```
git clone https://github.com/teraprath/PokeGuess.git
```
```
npm install
```
### Start Development Server
```
npm run dev
```
### Build
```
npm run build
```
